Core Courses
1 credit  ·  30 hours
Introduction to AI Product Development

Understand the landscape, meet the team, and ship your first AI-assisted website.

What students learn
How AI works AI limitations Tech team roles Prompt basics Build a website

This orientation course gives students a picture of the AI landscape before they write a single line of code. They'll learn how large language models actually work, where AI falls short, and what roles make up a modern software team. Finally, students put it all together by building and publishing their first AI-assisted website.

Course objectives
  • Explain how large language models generate output and where that process can go wrong
  • Identify the capabilities and real-world limitations of current AI tools
  • Describe the roles on a software product team and how they collaborate
  • Write clear, effective prompts to direct AI tools toward a specific outcome
  • Build and publish a basic website using AI-assisted development
3 credits  ·  90 hours
Software Systems and Architecture

Learn to read software before you build it, how systems are designed, wired together, and why.

What students learn
Computer architecture Data structures APIs & data flow System design Data privacy

Before students can build software confidently, they need to understand how it actually works under the hood. This course covers the conceptual foundations from how a computer processes instructions to how data moves across the internet. This foundation sets students up so they can make informed decisions when designing and building their own systems.

Course objectives
  • Describe how computer hardware components work together to run software
  • Identify common data structures and explain when each is appropriate
  • Trace a web request from browser to server and back, naming each step
  • Read and use a REST API, including authentication and JSON data handling
  • Recognize data privacy risks and apply basic mitigation strategies
4 credits  ·  120 hours
Software Development with AI

Go from understanding systems to building them, two real projects using AI as a core tool.

What students learn
Dev environment setup Write real code AI coding assistants Prompt engineering Evaluate AI output Ship projects

This is where students start writing real software. They set up a professional development environment, learn to code in a modern language, and use AI coding assistants as everyday tools rather than shortcuts. The course culminates in two complete projects, one guided and one independent, that students can add to a portfolio.

Course objectives
  • Configure a local development environment including terminal, editor, and version control
  • Use AI coding assistants to accelerate development
  • Craft precise prompts that produce accurate, usable code from an AI model
  • Review AI-generated code to verify it behaves as intended
  • Manage a project with Git and deliver two complete, deployable software projects
4 credits  ·  120 hours
AI Agents and Automation

Design and deploy agentic AI systems from no-code workflows to full agent frameworks.

What students learn
No-code automation APIs & webhooks Agent design Multi-step workflows Debug & test agents Responsible AI

Students move beyond writing code that runs once and start building systems that act on their own. Starting with no-code automation tools and progressing to full agent frameworks, this course teaches how to design AI-driven workflows that connect services, make decisions, and run with minimal human involvement.

Course objectives
  • Build multi-step automations using no-code tools that connect real services
  • Integrate external APIs and webhooks into automated workflows
  • Design an agentic system with clear goals, tools, and decision logic
  • Implement a working agent using a modern agent framework
  • Test, debug, and monitor an agent's behavior in a production-like environment
  • Evaluate an AI system for safety, bias, and unintended consequences
